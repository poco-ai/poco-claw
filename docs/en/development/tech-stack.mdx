---
title: Technology Stack
description: Core frameworks and tools used across the Poco project.
---

- **Frontend**: Next.js 16, React 19, TypeScript, Tailwind CSS v4, shadcn/ui
- **Backend services**: Python 3.12+, FastAPI, SQLAlchemy, Alembic, PostgreSQL
- **Executor**: Claude Agent SDK + FastAPI
- **Executor Manager**: APScheduler + httpx + FastAPI
- **Package managers**: `pnpm` for Node.js and `uv` for Python
